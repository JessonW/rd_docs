Error Code

[3000-5000]为AbleCloud内置的帐号管理，设备管理，存储服务等返回的错误码

[5001,6000]为AbleCloud平台返回的错误码

[6001-10000]为AbleCloud内部服务返回的错误码，因此建议用户自定义服务错误码区间为(1000-2000）或者10000以上。
